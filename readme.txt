== Description ==
Add different amount of points for WooCommerce Memberships purchases.

== Installation ==
1. Use WordPress\' built-in installer or
1. Upload the plugin files to the `/wp-content/plugins/mycred-woocommerce-memberships` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. A new Hook will appear in myCRED hooks menu
